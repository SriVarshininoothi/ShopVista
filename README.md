# 🛍️ ShopVista - E-commerce Website

ShopVista is a responsive e-commerce website built using **HTML, CSS, Bootstrap, and JavaScript**. It uses the **DummyJSON API** to display products dynamically with features like search, category filtering, sorting, and responsive design.

## 🚀 Live Demo

https://varshini-shopvista-site.vercel.app

---

## 📌 Features

- 🏠 Responsive Home Page
- 🎠 Bootstrap Carousel Banner
- 🛒 Dynamic Product Listing
- 🔍 Product Search
- 📂 Category Filter
- 📊 Sort Products
  - Price: Low to High
  - Price: High to Low
  - Top Rated
- ⭐ Dynamic Star Ratings
- 💰 Discount Percentage
- 📱 Fully Responsive Design
- 🎯 Featured Products Section
- 📦 Product Count Display
- ❌ No Products Found UI
- 🔄 Clear Filters Button
